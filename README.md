# DeepL Write for Raycast

Rewrite and improve text using the [DeepL Write API](https://developers.deepl.com/docs/api-reference/write).

## Features

- **Selected text**: Automatically replaces the selection with the improved version
- **Clipboard text**: Falls back to clipboard content and copies the result back

## Requirements

A [DeepL Pro](https://www.deepl.com/pro) subscription is required. The Write API is not available on the free tier.

## Setup

1. Install the extension
2. Get your API key from [DeepL Account](https://www.deepl.com/your-account/keys)
3. Enter the key when prompted on first run

## Usage

1. Select text in any application (or copy text to clipboard)
2. Open Raycast and run **Rewrite Text**
3. The improved text replaces your selection or is copied to clipboard
